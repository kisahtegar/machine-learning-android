# Draw Object at Canvas

Custom View and Canvas. With these two components, you can draw according to your needs in images resulting from machine learning processing.

| Nama Project                  | Draw Object at Canvas          |
|-------------------------------|--------------------------------|
| Target & Minimum Target SDK   | Phone and Tablet, API level 21 |
| Tipe Activity                 | Empty Views Activity           | 
| Activity Name                 | MainActivity                   |
| Language                      | Kotlin                         |
| Build Configuration Language  | Kotlin DSL                     |
